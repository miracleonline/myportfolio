Shortlink App

A full stack app built with React frontend and Nodejs backend. The app shortens url links and provides a shorter link which can still be used to access the original link.

Users can shorten any url link, view all shorten links and details about each such as the original link, the date created, number of visits via the app, device details etc.

Also users can search the list of shortened links using a live filtering search bar which starts filtering once they start typing and users can still decode the shorten link to view the original.

The frontend is deployed on Vercel and the backend on Railway with file-based storage(no database) while using GitHub for version control.
